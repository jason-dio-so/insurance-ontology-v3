## 삭제된 데이터 (참고용)

### 1차 삭제 (30개)
- 메타데이터: 개인형, 계약정보, 보험회사, 직업코드, 통합고객
- 문서구조: 가입담보 요약표, 전환가능계약
- 설명문: 순수보장성 상품으로..., 본 양식은 약관내용을...
- 플랜정보: 20년납 100세만기..., (30세|남|1급|...)
- 판매정보: 수도GA3-3지점, 이현기

- 메타데이터: 납입보험료, 무사고기간 1년
- 플랜유형: 1형(일반고지형), N형(건강고지형...)
- 설명문: 약관에서 정한..., 통풍진단담보의 보장개시일은...
- 안내문: 보험료 납입면제 안내, 상품별 보험료 할인 안내
